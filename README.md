# misc

[简体中文](README.zh.md) | [English](README.md)

A collection of small, miscellaneous tools and reference files.

## Contents

### `chinese/`

A small toolkit for working with the **GB2312** Chinese character encoding.

- `print2312.py` — Python script that generates a printable table of GB2312 zone-position codes.
- `gb2312.xlsx` — Reference spreadsheet of GB2312 characters.
- `gb2312_gbk.csv` — CSV table generated by `print2312.py`, containing all defined GB2312 code points encoded in GBK (with undefined zones/positions left blank).
- `gb2312_utf8.txt` — The same table as `gb2312_gbk.csv`, but converted to UTF-8 with commas replaced by spaces, making it easier to view or search in a plain-text editor.
